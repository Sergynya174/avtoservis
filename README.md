## Одностраничный сайт автосервиса

Проект для автосервиса в Москве.

#### Технологии:
* HTML5
* CCS3
* Методология БЭМ (наименования классов и файловая структура)
* Flexbox
* Анимации
* Трансформации
* Позиционирование
* Псевдоклассы
